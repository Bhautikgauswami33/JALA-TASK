Browser Compatibility Notes

  • Use vendor prefixes for older browser support (-webkit-, -moz-, -ms-, -o-)
  • Provide fallback values for newer CSS properties
  • Use feature detection with Modernizr when needed
  • Test on multiple browsers and versions
  • Use CSS reset or normalize.css for consistent base styles
  • Consider using autoprefixer tool to automate vendor prefixing
  • Progressive enhancement: start with basic functionality, add enhancements
